Projet .NET


Description

Application console permettant la gestion de contrôles des zones de stockages GSB, utilisant une architecture de couche de métier logiques.


Les utilisateurs de l'application peuvent consulter, ainsi que gérer les contrôles donc, ajouter, modifier ou supprimer.


Un contrôle est constitué d'un libellé,d'un résumé, d'une date, du nom de l'entreprise, d'un type de contrôle et le nom de la zone de stockage.


Objectifs

  1. Prendre connaissance d'un cahier des charges
  2. Proposer une solution
  3. Travailler en équipe
  4. Travailler en autonomie
  5. Présentation orale du projet

Pourquoi ce projet ?

  1. Prendre du savoir et du savoir-faire en liaison avec ma formation en BTS SIO
  2. Développer et appliquer mes compétences acquises durant ces 2 années de BTS

Outils utilisés

  1. Microsoft Visual Studio
  2. SQL server management studio
  3. GitHub, StarUML
